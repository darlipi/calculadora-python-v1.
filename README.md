# calculadora-python-v1
"Este es mi primer proyecto en Python. Es una calculadora básica por consola que permite realizar operaciones aritméticas fundamentales.

## 🚀 Características
* Soporta Suma, Resta, Multiplicación y División.
* **Control de errores:** Valida que no se pueda dividir entre cero.
* **Menú interactivo:** Si te equivocas de opción, el programa te lo advierte y vuelve a mostrar el menú usando un bucle `while`.

## 🛠️ Tecnologías utilizadas
* Python 3"
